# TNBF Website - Assets

This directory contains all static assets for the TNBF website.

## Directory Structure

### Images

- **hero/** - Hero section images for the homepage
- **industries/** - Industry-specific images
- **team/** - Team member photos
- **blog/** - Blog post images
- **testimonials/** - Customer testimonial images

### Icons

- Website icons, favicons, and UI icons

## Usage Guidelines

1. Optimize all images before adding them to the repository
2. Use descriptive filenames (e.g., `manufacturing-sector.jpg` instead of
   `img1.jpg`)
3. Recommended formats:
   - Photos: JPEG
   - Graphics/Logos: PNG or SVG
   - Icons: SVG when possible
4. Keep file sizes optimized for web (target < 200KB per image)
